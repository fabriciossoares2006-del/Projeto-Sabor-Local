Repositório criado para armazenar meus códigos de estudos, desafios de lógica e projetos desenvolvidos durante o 1° semestre da faculdade de Análise e Desenvolvimento de Sistemas (ADS).

## 🏪 Projeto em Destaque: Sabor Local (Sistema de Delivery)

O **Sabor Local** é um ecossistema completo de delivery voltado para o comércio gastronômico regional. Como o meu foco principal de carreira é o **Desenvolvimento Backend**, dediquei grande parte do projeto à modelagem de dados robusta e à estruturação lógica das regras de negócio.

### 🛠️ Stack Tecnológica Utilizada
* **Backend & Banco de Dados:** Xano (PostgreSQL)
* **Frontend:** FlutterFlow

### 🧠 Regras de Negócio & Fluxo do Usuário
O sistema foi projetado mapeando minuciosamente a jornada do usuário, incluindo:
* Fluxo de autenticação seguro (Cadastro/Login) com **Verificação de Código via E-mail**.
* Gerenciamento de múltiplos endereços e carrinho de compras dinâmico.
* Checkout estruturado para pagamentos via **PIX (com QR Code)** e Cartão de Crédito.
